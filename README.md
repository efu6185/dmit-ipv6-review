## 线路质量：CN2 GIA 和 CMIN2 的区别

这两条线路经常被混淆，说清楚比较重要。

**CN2 GIA**（Global Internet Access）是电信的顶级商用线路，去程回程都走 CN2，理论上是大陆方向延迟最低、最稳定的选择。DMIT 香港和洛杉矶 Premium 系列走的就是这条。

**CMIN2** 是移动的国际精品网，主要优化移动用户的访问体验。如果你的用户群体以移动宽带为主，CMIN2 节点可能比 CN2 GIA 更合适——移动直连，延迟表现同样出色。

我自己开了一个洛杉矶 Premium 的低配档跑了将近三周，主要用来跑一些定时任务和代理节点。下载速度稳定能跑到 150-200Mbps 区间，晚高峰没有明显掉速，这是我之前用过的几家里表现最稳的一次。

## 常见问题

### DMIT 的 IPv6 是原生 IP 吗？

大部分节点分配的是原生 IPv6 地址，不是广播 IP 或中转 IP。香港和洛杉矶节点的 IPv6 原生性较好，适合需要原生 IP 做特定平台访问的场景。具体可以在下单前通过工单确认。

### DMIT 支持哪些支付方式？

官网支持 PayPal、信用卡、支付宝以及加密货币（USDT 等）。支付宝对国内用户来说最方便，加密货币适合对隐私有要求的用户。

### DMIT 的套餐可以升级配置吗？

支持套餐升级，但降级通常不支持。如果不确定用量，建议先从低配档入手，跑一段时间再决定要不要升。

### 香港节点和洛杉矶节点哪个更适合大陆用户？

延迟优先选香港，香港 CN2 GIA 到大陆主要城市通常在 30-50ms。如果需要更大带宽或流量配额，洛杉矶 Eyeball（CMIN2）性价比更高，移动用户体验尤其好。

### DMIT 有没有退款保障？

官网提供一定期限内的退款政策，具体天数以购买时的条款为准。建议在购买前仔细阅读官网退款说明，避免误解。

### IPv6 地址分配是 /64 还是 /128？

不同套餐分配方式不同，多数套餐给 /64，部分给 /128。如果业务需要大段 IPv6 地址池，下单前通过工单确认是最稳妥的做法。

### DMIT 的 VPS 支持自定义 ISO 吗？

支持，可以通过控制面板挂载自定义 ISO 重装系统。官方也提供常见 Linux 发行版的一键安装，操作不复杂。
